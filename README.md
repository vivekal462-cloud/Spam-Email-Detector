# Spam Email Detector

A Machine Learning project that detects whether an email/message is Spam or Ham using Python and Scikit-Learn.

## Features

* Spam Email Detection
* TF-IDF Vectorization
* Naive Bayes Classifier
* Confidence Score Prediction
* Tkinter GUI

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Tkinter

## Project Structure

```text
Spam-Email-Detector/
│
├── spam_detector.py
├── gui.py
├── emails.csv
└── LICENSE
```

## How to Run

```bash
pip install pandas scikit-learn
python gui.py
```
